## Key findings

- **Matrix LR 0.06 > 0.04**: Higher Muon LR gave a meaningful improvement (−0.041 val_bpb)
- **Smaller batch = more steps = better in 5 min**: Halving batch size doubled optimizer steps within the same time budget, leading to the best result so far
- **Deeper model hurt**: Going from 8→10 layers increased params to 85.9M but reduced step count too much for the 5-min budget
